프로젝트 개요

사이트 바로 가기
    프로젝트명
  • 필프레임 자사몰 리뉴얼
    기여도
  • - 60%
    기간
  • - 5개월
    인원
  • - 개발자 2명
  • - 퍼블리셔 1명
  • - 웹디자이너 1명
    담당 업무
  • - XD를 활용한 UI/UX 디자인
  • - 상품 상세페이지 디자인
  • - 메인, 카테고리, 이벤트 등 각종 배너 제작
  • - 공동구매 페이지 퍼블리싱
  • - 기업구매 페이지 퍼블리싱
  • - 관리자 페이지 디자인 및 유지보수
  • - 운영 중 콘텐츠 및 이벤트 페이지 제작
    프로젝트 환경
  • - 자사몰
  • - HTML / CSS / JavaScript (Atom)
  • - Adobe XD
  • - Photoshop
  • - illustrator
    주요 기여
  • - 메인 페이지 및 상품 상세페이지 리뉴얼 디자인 진행
  • - 공동구매 및 기업구매 페이지 퍼블리싱 담당
  • - 프로모션 배너 및 이벤트 콘텐츠 제작
  • - 운영 유지보수 및 디자인 개선 작업 수행
    프로젝트 회고
  • - 디자인부터 퍼블리싱까지 직접 담당하며 화면 설계와 구현 과정을 모두 경험할 수 있었음.
  • - 클래스 및 아이디 네이밍 규칙의 중요성을 체감하며 유지보수성을 고려한 코딩 습관을 익힘.
  • - 다양한 디바이스 환경을 고려한 반응형 웹 구현의 필요성을 경험함.
  • - 심미성뿐 아니라 사용자의 편의성과 가독성을 고려한 UI/UX 설계의 중요성을 이해하게 됨.
